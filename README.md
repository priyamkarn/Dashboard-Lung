Project Lung

Decision Support System for EV Bus Dispatch Operations.

Tech Stack

* Next.js 16
* React 19
* TypeScript
* Tailwind CSS v4
* shadcn/ui
* Supabase
* Drizzle ORM
* Better Auth
* pnpm

Getting Started

Install dependencies:

pnpm install

Start development server:

pnpm dev

Build project:

pnpm build
